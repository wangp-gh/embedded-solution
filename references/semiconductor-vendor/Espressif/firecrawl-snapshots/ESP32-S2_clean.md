## ESP32-S2 Features
ESP32-S2 is a highly integrated, low-power, single-core Wi-Fi Microcontroller SoC,
designed to be secure and cost-effective, with a high performance and a rich set of IO capabilities.
![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-s2/security.png)
Unparalleled Security for Your Connected Devices
- RSA-3072-based secure boot
- AES-128/192/256-XTS-based flash encryption
- Protected private key and device secrets from software access
- Cryptographic accelerators for enhanced performance
- Protection against physical fault injection attacks
![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-s2/rich.png)
Display, Touch Capabilities and Rich IO
ESP32-S2 integrates a rich set of peripherals, with 43 programmable GPIOs which can be flexibly configured to provide USB OTG, LCD interface, camera interface, SPI, I2S, UART, ADC, DAC and other common functionality. With LCD interface and 14 configurable capacitive touch GPIOs, ESP32-S2 provides the optimal HMI solution for touchscreen and touchpad-based devices.
![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-s2/temperatures.png)
Solid Wi-Fi Performance at Extreme Temperatures
ESP32-S2’s operating temperature ranges from -40 to +105 degrees
Celsius, thus facilitating a variety of industrial, consumer and lighting applications.

## Display and Touch HMI
Both ESP32 and ESP32-S2 provide a touchscreen and capacitive touchpad type functionality
in the hardware, as well as easy-to-use software libraries. This makes ESP32- and ESP32-S2-based
designs an excellent choice for touch modules with great connectivity, which can be used in home appliances and other types of devices.
![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-s2/touch-hmi.png)
