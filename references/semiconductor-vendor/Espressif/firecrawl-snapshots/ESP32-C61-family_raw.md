The ESP32-C61 is designed to meet the growing demand for advanced Wi-Fi 6 technology, offering optimized peripherals, enhanced connectivity, and expanded memory options. This next-generation SoC integrates Wi-Fi 6 and Bluetooth 5 (LE), providing a powerful and versatile solution for your connectivity needs.


## Features

![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/wireless-connectivity.png)

### Wireless Connectivity

The Wi-Fi 6 support is optimized for IoT devices, and ESP32-C61 supports a 20MHz bandwidth for 802.11ax mode, and a 20/40MHz bandwidth for 802.11b/g/n mode. The 802.11ax mode supports features like OFDMA, and MU-MIMO, which provide reliable and low-latency connection, along with Target-Wake-Time, which is the basis of ultra-low power applications. Additionally, Bluetooth 5 (LE) radio supports long-range operation through advertisement extension and coded PHY, while also supporting 2 Mbps of high throughput PHY. ESP32-C61 also supports the [Bluetooth Mesh 1.1 protocol](https://www.espressif.com/en/news/Bluetooth_Mesh_1.1).


![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/system-and-momory.png)

### System and Memory

ESP32-C61 has a single-core, 32-bit RISC-V microcontroller that can be clocked up to 160 MHz. It has a 320KB on-chip SRAM, a 256KB ROM, and works with Quad SPI flash. The SoC includes support for in-package PSRAM with Quad SPI of up to 120MHz, thus enabling developers to focus on application development without worrying about memory optimization.


![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/security.png)

### Security

ESP32-C61, strongly emphasizes affordable security, incorporating features such as secure boot, flash and PSRAM encryption, and cryptographic accelerators. It also has an ECDSA-based Digital Signature Peripheral in hardware, thus safeguarding private keys from software access. The addition of a Trusted Execution Environment (TEE), using an Access Permission Management (APM) hardware block and Physical Memory Protection (PMP), provides mechanisms for software isolation.


![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/optimized-peripherals.png)

### Optimized Peripherals

ESP32-C61 encompasses standard microcontroller peripherals, such as I2C, I2S, SPI, UART, LED PWM, ADC, GPIO, LP IO, Timers, and GDMA. Specialized peripherals include the Event Task Matrix (ETM) for automation-triggered tasks and the Analog Voltage Comparator for easy zero-crossing detection.


![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/software-availability.png?v=1)

### Software Availability

ESP32-C61 is supported through Espressif’s mature IoT Development Framework ( [ESP-IDF](https://www.espressif.com/en/products/sdks/esp-idf)), so that our customers can benefit from their familiarity with our field-proven platform that already powers millions of connected devices. The [ESP-Matter SDK](https://github.com/espressif/esp-matter) includes support for creating Matter-enabled products with ESP32-C61. With the inclusion of PSRAM, the ESP32-C61 facilitates ease of development and supports a wide range of applications. For customers interested in utilizing the ESP32-C61 as a communication co-processor with an external host, Espressif offers both [ESP-Hosted](https://github.com/espressif/esp-hosted) and [ESP-AT](https://github.com/espressif/esp-at) firmware.
