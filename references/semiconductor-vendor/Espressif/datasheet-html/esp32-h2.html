[Buy Now](https://www.espressif.com/en/products/socs/esp32-h2#buy-now) ESP32-H2 is a low-power single-core 32-bit RISC-V SoC that integrates IEEE 802.15.4 and Bluetooth® 5 (LE) for secure, energy-efficient wireless connectivity. It enables reliable low-power operation across a broad range of wireless endpoints. ESP32-H2 includes 320 KB SRAM, 128 KB ROM, 4 KB LP memory, and built-in 2 MB or 4 MB SiP flash, along with 19 programmable GPIOs supporting ADC, SPI, UART, I2C, I2S, RMT, GDMA, and LED PWM. ## Features ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-h2/integrated-ieee.png) ### Integrated IEEE 802.15.4 Thread and Zigbee protocol supported ESP32-H2 combines IEEE 802.15.4 radio connectivity, which is vital to mesh architectures with low power consumption. It also makes the Thread and Zigbee protocols available, thus addressing a variety of cases in application development. ESP32-H2 has been certified as a [“Zigbee-Compliant Platform”](https://www.espressif.com/sites/default/files/ESP32-H2%20Zigbee%20Compliant%20Platform%20Certification.pdf) and has officially become a [“Thread-Certified 1.3.0 Component”](https://www.espressif.com/sites/default/files/ESP32-H2%20Thread%20V1.3.0%20Interoperability%20Certification.pdf). ### Enabling Matter Thread end devices, Thread border router, Matter gateway ESP32-H2’s concurrent support for IEEE 802.15.4 and Bluetooth LE enables customers to build Matter-compliant Thread end-point devices which ensure interoperability among smart-home devices from different systems. By combining ESP32-H2 and another Espressif Wi-Fi SoC, a Thread border router or a Matter-Zigbee bridge can be built for connecting Thread or Zigbee devices onto Matter Wi-Fi networks. ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-h2/enabling-matter.png) ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-h2/robust-security.png) ### Robust Security Hardware security features ESP32-H2 can facilitate building securely connected devices through such hardware features as ECC-based secure boot, AES-128/256-XTS-based flash encryption, digital signature with ECDSA support, an HMAC peripheral for identity protection, as well as cryptographic accelerators for improved performance. With such robust security features, all connected devices can be secured at an affordable cost for users' complete peace of mind. ### Software Availability Supported by ESP-IDF and other Espressif SDK ESP32-H2 is supported through Espressif’s mature IoT Development Framework (ESP-IDF), so that our customers can benefit from their familiarity with our field-proven platform that already powers millions of connected devices. ESP32-H2 supports a production-ready [ESP-Zigbee-SDK](https://github.com/espressif/esp-zigbee-sdk) to build ready-to-use Standard Home automation devices. ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-h2/software-availability.png) ## Products & Resources ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-series.png) -
