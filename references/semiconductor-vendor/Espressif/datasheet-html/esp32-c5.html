[Buy Now](https://www.espressif.com/en/products/socs/esp32-c5#buy-now) ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c5/esp32-c5-video-cover-en.png) Espressif’s ESP32-C5 is the industry’s first RISC-V MCU that supports 2.4 and 5 GHz dual-band Wi-Fi 6, along with Bluetooth 5 (LE) and IEEE 802.15.4 (Zigbee, Thread) connectivity. It is designed for applications that require high-efficiency wireless transmission. ESP32-C5 has a 32-bit single-core processor which can clock up to 240 MHz in speed. It has a 384 KB on-chip SRAM along with external PSRAM support, 320 KB of ROM. It has up to 29 programmable GPIOs, supporting all the commonly used peripherals and the best-in-class security features. ## Features ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c5/feature-icon-1.png) ### 2.4 and 5 GHz Dual-band Support The 5 GHz band offers customers more reliable and low-latency wireless connectivity, ideal for scenarios requiring minimal traffic and lower interference for critical applications. Dual band support enables users to assign different IoT devices to specific networks based on their importance. By incorporating dual-band connectivity into their devices, customers can achieve maximum flexibility and future-proof their solutions. ### Wi-Fi 6 ESP32-C5 features a dual-band Wi-Fi 6 (802.11ax) radio with 802.11a/b/g/n/ac backward compatibility. The 802.11ax standard in ESP32-C5 offers various benefits for IoT. Key features include Target Wake Time (TWT) for extended device sleep, MU-MIMO, and OFDMA for increased device capacity and lower latency. BSS coloring ensures stable connectivity even in congested areas with multiple SSIDs. ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c5/feature-icon-2.png) ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c5/feature-icon-3.png?v=1) ### Outstanding RF Performance The ESP32-C5 MCU supports Bluetooth 5 (LE), Zigbee, and Thread. It includes an IEEE Standard 802.15.4 subsystem that integrates PHY and MAC layers and supports various software stacks such as Thread, Zigbee, Matter, HomeKit, and MQTT. Additionally, ESP32-C5 continues to support Bluetooth LE SIG Mesh and [ESP-Mesh-Lite](https://www.espressif.com/en/sdks/esp-mesh-lite), with Bluetooth 5 (LE) enabling long-range operation through advertising extensions and coded PHY. It also supports 2 Mbps of high throughput PHY to help build devices with great coverage and improved usability. ### Security Aligning with Espressif’s philosophy, the ESP32-C5 prioritizes affordable security, featuring secure boot, flash and PSRAM encryption, and cryptographic accelerators. It includes a hardware-based Digital Signature Peripheral and a dedicated Key Manager for secures key storage and deployment. The Trusted Execution Environment (TEE), with an Access Permission Management (APM) hardware block and Physical Memory Protection (PMP), enhances application security management flexibility. ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c5/feature-icon-5.png) ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c5/feature-icon-4.png) ### Software Availability ESP32-C5 is supported on Espressif’s open-source [ESP-IDF](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/), which already powers millions of devices in the field. This support ensures the availability of a robust SDK and tools, as well as an easy application migration path for developers. The added support for PSRAM enables developers to focus on application development without worrying about memory optimization. The larger memory availability also provides better futureproofing for upgrades. For customers who would like to use ESP32-C5 as a communication co-processor with an external host, [ESP-AT](https://github.com/espressif/esp-at) and [ESP-Hosted](https://github.com/espressif/esp-hosted) are also available. ## Products & Resources ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-series.png) - ESP32-C5 [See More SoCs](https://www.espressif.com/en/products/socs?id=ESP32-C5) ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-modules.png) - ESP32-C5-WROOM-1 [See More Modules](https://www.espressif.com/en/products/modules?id=ESP32-C5) ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-devkits.png) - [ESP32-C5-DevKitC-1](https://docs.espressif.com/projects/esp-dev-kits/en/latest/esp32c5/esp32-c5-devkitc-1/index.html) [See More DevKits](https://www.espressif.com/en/products/devkits?id=ESP32-C5) - ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-1.png)[Documents >](https://www.espressif.com/en/support/documents/technical-documents) - ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-2.png)[Product Change Notifications >](https://www.espressif.com/en/support/documents/pcns) - ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-3.png)[Certificates >](https://www.espressif.com/en/support/documents/certificates) - ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c61/icon-4.png)[Advisories >](https://www.espressif.com/en/support/documents/advisories) ## ![](https://www.espressif.com/sites/all/themes/espressif/images/esp32-c3/esp-product-selector-logo.png)ESP Product Selector Choosing the ESP products you need has never been easier!
