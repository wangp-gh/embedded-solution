#### Overview
The CH32V203 series is an industrial-grade, enhanced low-power general-purpose microcontroller based on a 32-bit RISC-V core. It delivers high-performance with a maximum system clock frequency of 144MHz and low power consumption, operating at as low as 49.3μA/MHz. The CH32V203 integrates dual USB interfaces supporting both USB Host and USB Device functionality. It features one CAN interface (CAN 2.0B active), dual OPAs, 4 UARTs, dual I2C interfaces, 12-bit ADC, and 10-channel Touchkey. It also provides rich peripheral resources and is available in multiple packages including TSSOP20, QFN28, QFN48X7, LQFP32, and LQFP48 packages for flexible customer selection.

#### Features
- QingKe V4B processer, up to 144MHz system frequency
- Support single-cycle multiplication and hardware division
- 20KB SRAM，64KB CodeFlash
- Supply voltage range: 2.5V/3.3V, GPIO unit with independent power supply
- Low power consumption mode: Sleep/Stop/Standby
- Operating mode as low as 49.3 μA/MHz
- Sleep mode as low as 19.4 μA/MHz
- Power-on/down reset (POR/PDR) , Programmable Voltage Detector (PVD)
- 1×8-channel DMA controller
- 2 sets of OPA
- 10-channel 12-bit ADC conversion, 10-channel TouchKey
- 4 sets of timers
- 1 USB2.0 full-speed host/device interface
- 1 USB2.0 full-speed device interface
- 4 USART interfaces
- 1 CAN interface (2.0B active)
- 2 sets of I2C interface, 2 sets of SPI interface
- 37 I/O pins, all of which can be mapped to 16 external interrupts
- CRC calculation unit, 96-bit unique ID
- 2-wire serial debugging modes
- Package form:  TSSOP20, QFN28, QFN48X7, LQFP32, LQFP48
