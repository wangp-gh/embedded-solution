#### Overview
CH32V305/7 series are interconnected microcontrollers based on 32-bit RISC-V design, equipped with hardware stack area, fast interrupt entry, which greatly improves interrupt response speed based on standard RISC-V. Adding single-precision floating-point instruction set and expanding the stack area provides higher computing performance. Expand the number of UART to 8 groups and motor timers to 4 groups. Provides USB2.0 high-speed interface (480Mbps) and built-in PHY transceiver, upgraded Ethernet MAC to Gigabit and integrated 10M-PHY module.
#### System Block Diagram
![System Block Diagram](http://img.wch.cn/20240122/2e6dfc3c-9568-4bd6-8483-67e876c89a0f.png)
#### Features
- QingKe V4F processor, up to 144MHz system frequency
- Support single-cycle multiplication and hardware division, support hardware floating point operation (FPU)
- 64KB SRAM, 256KB Flash
- Supply voltage: 2.5/3.3V, GPIO unit independent power supply
- Multiple low-power modes: Sleep/Stop/Standby
- Power on/off reset, programmable voltage monitor
- 2 sets of 18-channel general-purpose DMA
- 4 sets of OPAs
- 1 TRNG
- 2 sets of 12-bit DAC conversion
- 2 units of 16-channel 12-bit ADC conversion, 16-channel touch key TouchKey
- 10 sets of timers
- USB2.0 full-speed OTG interface
- USB2.0 high-speed host/device interface (480Mbps built-in PHY)
- 3 USART interfaces and 5 UART interfaces
- 2 CAN interfaces (2.0B active)
- SDIO interface, FSMC interface, DVP digital image interface
- 2 sets of I2C interfaces, 3 sets of SPI interfaces, 2 sets of I2S interfaces
- Gigabit Ethernet Controller ETH (built-in 10M PHY)
- 80 I/O ports, can be mapped to 16 external interrupts
- CRC calculation unit, 96-bit chip unique ID
- 2-wire serial debug interface
- Package: LQFP64M, LQFP100
