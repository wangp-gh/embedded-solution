#### Overview
The CH32V303 series is an industrial-grade general-purpose microcontroller based on a 32-bit RISC-V design, featuring a hardware stack area and fast interrupt entry points that significantly enhance interrupt response speed compared to standard RISC-V implementations. The CH32V303 series incorporates the V4F core. It adds a single-precision floating-point instruction set, expands the stack area for higher computational performance, and increases the number of UARTs to 8 and motor timers to 4.
#### System Block Diagram
![System Block Diagram](https://img.wch.cn/20231023/aa3b221a-871f-49c7-b769-afa703f3573d.png)
#### Features
- QingKe V4F processer, up to 144MHz system frequency
- Support single-cycle multiplication and hardware division, support hardware floating-point unit (FPU)
- 64KB SRAM，256KB CodeFlash
- Supply voltage range: 2.5V/3.3V, GPIO synchronous supply voltage
- Low power consumption mode: Sleep/Stop/Standby
- Power-on/down reset (POR/PDR) , Programmable Voltage Detector (PVD)
- 2×18-channel DMA controller
- 4 sets of OPA
- 1 TRNG
- 2 units 16-channel 12-bit ADC conversion, 16-channel TouchKey
- 10 sets of timers
- 1 USB2.0 FS host/device interface
- 3 USART interfaces and 5 UART interfaces
- 1 CAN interface (2.0B active)
- SDIO interface, FSMC interface
- 2 sets of I2C interface, 3 sets of SPI interface, 2 sets of I2S interface
- 80 I/O pins, all of which can be mapped to 16 external interrupts
- CRC calculation unit, 96-bit unique ID
- 2-wire serial debugging modes
- Package form:  LQFP48, LQFP64M, LQFP100
