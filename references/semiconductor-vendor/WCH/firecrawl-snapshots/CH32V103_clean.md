#### Overview
The CH32V103 series is a 32-bit general-purpose microcontroller based on the RISC-V3A processor, which is designed using the open-source RISC-V instruction set. On-chip integration includes clock security mechanisms, multi-level power management, and a general-purpose DMA controller. This series features rich peripheral resources such as one USB 2.0 host/device interface, multi-channel 12-bit ADC conversion modules, multi-channel TouchKey, multiple timer groups, and multiple I2C/USART/SPI interfaces.

#### Features
- QingKe V3A processer, up to 80MHz system frequency
- Support single-cycle multiplication and hardware division
- 20KB SRAM，64KB CodeFlash
- Supply voltage range: 2.7V~5.5V, GPIO synchronous supply voltage
- Low power consumption mode: Sleep/Stop/Standby
- Power-on/down reset(POR/PDR)
- Programmable Voltage Monitor (PVD)
- 7-channel DMA controller
- 16 TouchKey channel detection
- 16×12-bit ADC conversion channels
- 7 timers
- 1 USB 2.0 host/device interface (full-speed and low-speed)
- 2 I2C interfaces (supporting SMBus/PMBus)
- 3 USART interfaces
- 2 SPI interfaces (supporting Master and Slave modes)
- 51 I/O pins, all of which can be mapped to 16 external interrupts
- CRC calculation unit, 96-bit unique ID
- 2-wire serial debugging modes
- Package form:  LQFP64M,  LQFP48, QFN48X7
