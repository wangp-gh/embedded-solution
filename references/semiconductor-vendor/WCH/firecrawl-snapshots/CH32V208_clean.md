#### Overview
The CH32V208 series is a wireless microcontroller based on the 32-bit RISC-V architecture, featuring a hardware stack area and fast interrupt entry point that significantly enhance interrupt response speed compared to standard RISC-V designs. Equipped with the V4C core, it incorporates a memory protection unit while reducing hardware division cycles, enabling zero-wait operation at a 144MHz clock frequency. On-chip integration includes a 2Mbps Bluetooth Low Energy BLE communication module, 10M Ethernet MAC+PHY module, USB 2.0 full-speed device + host/device interface, and a CAN controller.

![System Block Diagram](https://img.wch.cn/20250522/639e6803-d28f-408c-8b72-053e1ca4405f.jpg)
#### Features
- QingKe V4C processer, up to 144MHz system frequency
- Support single-cycle multiplication and hardware division
- 64KB SRAM，128KB Flash
- Supply voltage range: 2.5V/3.3V, GPIO unit with independent power supply
- Low power consumption mode: Sleep/Stop/Standby
- Bluetooth Low Energy BLE 5.3
- 10M Ethernet controller (MAC+PHY)
- Power-on/down reset (POR/PDR) , Programmable Voltage Detector (PVD)
- 2 sets of OPA
- 16-channel 12-bit ADC conversion, 16-channel TouchKey
- 5 sets of timers
- USB2.0 full-speed host/device+device interface
- 4 USART interfaces
- 1 CAN interface (2.0B active)
- 2 sets of I2C interface, 2 sets of SPI interface
- 53 I/O pins, all of which can be mapped to 16 external interrupts
- CRC calculation unit, 96-bit unique ID
- 2-wire serial debugging modes
- Package form:  LQFP64M, QFN68, QFN48, QFN28
