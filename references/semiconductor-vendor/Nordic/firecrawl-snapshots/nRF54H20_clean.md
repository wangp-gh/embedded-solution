# nRF54H20   System-on-Chip
## Revolutionary multiprotocol SoC
nRF54H20 is a compact ultra-low power SoC with superior processing power, a generous amount of memory, and excellent efficiency. It has a new 4th generation multiprotocol 2,4 GHz radio and isolated, physically protected advanced system security. Its unique combination of features, all integrated into a compact SoC, will enable developers to build innovative IoT products that haven't been possible before.
### Main benefits
- Reduces design size: Replace multiple components with a highly integrated SoC requiring minimal external components.
- Prolongs battery life or reduces battery size: Efficient processing, ultra-low power radio, and minimal sleep currents.
- Provides long range: Best-in-class multiprotocol radio with 10 dBm TX power, -99 dBm RX sensitivity for Bluetooth LE and -104 dBm for 802.15.4.
- Protects against security threats: Isolated, physically protected advanced system security with secure boot, secure firmware update, secure storage, and cryptographic acceleration.

### Discover nRF54H20
##### Superior processing power and a generous amount of memory
nRF54H20 integrates multiple Arm Cortex-M33 processors and RISC-V coprocessors, clocked up to 320 MHz. Each processor is optimized for a specific type of workload. The processing power of the application processor is doubled compared to nRF5340. It has a generous amount of integrated memory with 2 MB of non-volatile memory and 1 MB of RAM, consuming significantly less power than external memory. The processing power and amount of memory make it perfect for running complex applications, also including machine learning (ML) models and sensor fusion at the edge.
##### World-leading processing efficiency
nRF54H20 offers a unique combination of processing efficiency and performance. The processing efficiency of nRF54H20's application processor has been proven with the EEMBC ULPMark®-CoreMark, which benchmarks the processor configured either for maximum processing efficiency or performance, using CoreMark as the workload. The nRF54H20’s application processor achieved the following scores: ULPMark-CM score of 170 with 515 CoreMark (configured for maximum processing efficiency), and ULPMark-CM score of 132 with 1290 CoreMark (configured for maximum processing performance).
##### New exciting digital and analog interfaces
The SoC has several new exciting digital and analog interfaces, including a high-speed USB (480 Mbps), a CAN FD controller, and a 14-bit ADC. This is in addition to numerous other analog and digital interfaces.
##### 4th generation 2.4 GHz radio
The new 4th generation nRF54H Series multiprotocol radio provides -99 dBm RX sensitivity for 1 Mbps Bluetooth Low Energy and -104 dBm for 802.15.4 protocols. Combined with up to 10 dBm TX power, it offers improved robustness and longer range. nRF54H20 supports Bluetooth Core 6.0 features and is designed to support upcoming Bluetooth specifications. It is designed to support Bluetooth Low Energy, Thread, Matter, and 2.4 GHz proprietary protocols with throughput up to 4 Mbps to enable ultra-low-latency applications. The radio RX current is as low as 1.7 mA (3V, DC/DC), resulting in major energy savings, especially for applications that are mainly receivers, such as earbuds and wearables.
##### State-of-the-art security
With the nRF54H Series, we prioritized security with the design focused on isolation of security features and physical protected. nRF54H20 is designed for PSA Certified Level 3, the highest level in the PSA Certified IoT security standard. It supports security services such as Secure Boot, Secure Firmware Update, and Secure Storage. The cryptographic accelerators are hardened against side-channel attacks, while tamper sensors can detect an attack in progress and take appropriate action.
##### All-in-one solution
nRF54H20 is a compact all-in-one solution that can replace multiple components on the PCB, reducing design size. For example, an application MCU, an external flash, and a wireless SoC can be replaced with a single compact nRF54H20. In addition, its excellent energy efficiency enables smaller batteries to be used, further reducing both the design size and cost.
